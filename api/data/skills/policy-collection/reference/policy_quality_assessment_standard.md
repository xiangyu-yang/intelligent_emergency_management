# 应急政策文件质量评估标准

## 一、概述

本标准规定了应急政策文件收集过程中的质量评估方法、评分标准和评估流程，确保收集的政策文件具备高质量、高准确性和高可用性。

## 二、质量评估体系

### 2.1 评估维度

应急政策文件质量评估采用五维评估体系：

```
┌─────────────────┐
│   完整性评估    │ (Completeness)
└─────────────────┘
         ↓
┌─────────────────┐
│   准确性评估    │ (Accuracy)
└─────────────────┘
         ↓
┌─────────────────┐
│   时效性评估    │ (Timeliness)
└─────────────────┘
         ↓
┌─────────────────┐
│   可用性评估    │ (Usability)
└─────────────────┘
         ↓
┌─────────────────┐
│   合规性评估    │ (Compliance)
└─────────────────┘
```

### 2.2 综合质量评分公式

**综合质量评分 = (完整性评分×0.25) + (准确性评分×0.25) + (时效性评分×0.20) + (可用性评分×0.20) + (合规性评分×0.10)**

- **优秀**：综合评分 ≥ 90分
- **良好**：综合评分 ≥ 80分
- **合格**：综合评分 ≥ 70分
- **不合格**：综合评分 < 70分

## 八、综合质量评估流程

### 8.1 评估流程图

```
┌─────────────┐
│  文件收集   │
└──────┬──────┘
       ↓
┌─────────────┐
│ 自动化评估  │（完整性、准确性）
└──────┬──────┘
       ↓
┌─────────────┐
│ 人工审核    │（时效性、可用性、合规性）
└──────┬──────┘
       ↓
┌─────────────┐
│ 综合评分    │（计算综合质量评分）
└──────┬──────┘
       ↓
┌─────────────┐
│  分级入库   │（优秀、良好、合格、不合格）
└──────┬──────┘
       ↓
┌─────────────┐
│  后续跟踪   │（定期复评、质量提升）
└─────────────┘
```

### 8.2 自动化评估步骤

#### 步骤1：完整性自动检查
- 检查必填字段是否完整
- 检查文件附件是否齐全
- 检查元数据是否完整

#### 步骤2：准确性自动检查
- 对比文件内容与原文
- 验证元数据准确性
- 验证分类准确性

#### 步骤3：生成初步评分
- 计算完整性评分
- 计算准确性评分
- 标记待人工审核项

### 8.3 人工审核步骤

#### 步骤1：时效性审核
- 确认文件发布时间
- 确认文件生效状态
- 确认文件更新情况

#### 步骤2：可用性审核
- 测试文件可打开性
- 测试内容可读性
- 验证检索功能

#### 步骤3：合规性审核
- 验证来源合法性
- 验证内容合规性
- 验证格式合规性

#### 步骤4：综合评分计算
- 根据公式计算综合质量评分
- 确定文件质量等级
- 提出改进建议

### 8.4 分级入库流程

#### 优秀文件（≥90分）
- 直接入库，标记为"优秀"
- 优先检索推荐
- 定期复评

#### 良好文件（≥80分）
- 入库，标记为"良好"
- 正常检索
- 定期复评

#### 合格文件（≥70分）
- 入库，标记为"合格"
- 标记需改进项
- 定期复评，跟踪改进

#### 不合格文件（<70分）
- 暂不入库，标记为"不合格"
- 记录不合格原因
- 重新收集或补充完善

## 九、质量提升措施

### 9.1 不合格文件处理

#### 处理流程
```
不合格文件 → 分析原因 → 补充完善 → 重新评估 → 再次入库
```

#### 处理措施
- **完整性不足**：补充缺失信息
- **准确性不足**：更正错误信息
- **时效性不足**：更新最新版本
- **可用性不足**：优化格式编码
- **合规性不足**：调整合规问题

### 9.2 定期质量复评

#### 复评频率
- 优秀文件：每季度复评
- 良好文件：每季度复评
- 合格文件：每月复评

#### 复评内容
- 检查文件状态变化
- 检查新版本发布
- 检查关联文件更新
- 重新计算质量评分

### 9.3 质量提升策略

#### 策略1：源头质量控制
- 选择权威来源渠道
- 使用标准化收集工具
- 配置自动化质量检查

#### 策略2：过程质量控制
- 建立标准化收集流程
- 实施多级审核机制
- 记录质量问题日志

#### 策略3：结果质量控制
- 建立质量评分体系
- 实施分级入库机制
- 建立质量跟踪机制

# completeness_checker.py
class CompletenessChecker:
    def check_basic_info(self, policy):
        score = 0
        if policy.title: score += 15
        if policy.document_number: score += 10
        if policy.issue_date: score += 10
        if policy.issuing_authority: score += 5
        return score
    
    def check_content(self, policy):
        score = 0
        if policy.content: score += 20
        if policy.attachments: score += 10
        return score
    
    def check_metadata(self, policy):
        score = 0
        if policy.category_level1: score += 10
        if policy.keywords: score += 10
        if policy.summary: score += 5
        if policy.tags: score += 5
        return score
    
    def calculate_score(self, policy):
        basic_score = self.check_basic_info(policy)
        content_score = self.check_content(policy)
        metadata_score = self.check_metadata(policy)
        
        completeness_score = (
            basic_score * 0.4 +
            content_score * 0.3 +
            metadata_score * 0.3
        )
        return completeness_score
```

class AccuracyChecker:
    def check_content_accuracy(self, policy, original_content):
        if policy.content == original_content:
            return 30
        elif self.similarity_check(policy.content, original_content) > 0.8:
            return 15
        else:
            return 0
    
    def check_metadata_accuracy(self, policy, original_metadata):
        score = 0
        if policy.title == original_metadata['title']:
            score += 10
        if policy.document_number == original_metadata['document_number']:
            score += 10
        if policy.issue_date == original_metadata['issue_date']:
            score += 10
        return score
    
    def check_classification_accuracy(self, policy):
        score = 0
        if self.validate_category(policy.category_level1):
            score += 8
        if self.validate_category(policy.category_level2):
            score += 6
        if self.validate_category(policy.category_level3):
            score += 6
        return score
    
    def calculate_score(self, policy, original_data):
        content_score = self.check_content_accuracy(
            policy, original_data['content']
        )
        metadata_score = self.check_metadata_accuracy(
            policy, original_data['metadata']
        )
        classification_score = self.check_classification_accuracy(policy)

# quality_scorer.py
class QualityScorer:
    def calculate_comprehensive_score(
        self,
        completeness_score,
        accuracy_score,
        timeliness_score,
        usability_score,
        compliance_score
    ):
        comprehensive_score = (
            completeness_score * 0.25 +
            accuracy_score * 0.25 +
            timeliness_score * 0.20 +
            usability_score * 0.20 +
            compliance_score * 0.10
        )
        return comprehensive_score
    
    def determine_quality_level(self, comprehensive_score):
        if comprehensive_score >= 90:
            return '优秀'
        elif comprehensive_score >= 80:
            return '良好'
        elif comprehensive_score >= 70:
            return '合格'
        else:
            return '不合格'
    
    def generate_quality_report(self, policy, scores):
        report = {
            'policy_id': policy.id,
            'policy_title': policy.title,
            'completeness_score': scores['completeness'],
            'accuracy_score': scores['accuracy'],
            'timeliness_score': scores['timeliness'],
            'usability_score': scores['usability'],
            'compliance_score': scores['compliance'],
            'comprehensive_score': scores['comprehensive'],
            'quality_level': scores['quality_level'],
            'evaluation_time': datetime.now(),
            'evaluator': 'system'
        }
        return report
```
